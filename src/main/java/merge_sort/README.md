# Merge sort
## Exercise description
Merge sort is a sorting algorithm invented by John von Neumann. 

For how it works please read the Wikipedia article about it: https://en.wikipedia.org/wiki/Merge_sort

Also, here are 2 visualizations about it: 
* http://algo-visualizer.jasonpark.me/#path=sorting/merge/bottom_up
* https://www.hackerearth.com/practice/algorithms/sorting/merge-sort/visualize/

## Execution
mergeSort() method sorts any given array using merge-sort recursive algorithm.
